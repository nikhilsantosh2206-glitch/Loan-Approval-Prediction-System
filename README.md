# 🏦 Loan Approval Prediction System

A Machine Learning web application that predicts whether a loan application is likely to be approved or rejected based on applicant details. The project is built using Python, Scikit-learn, and Streamlit, providing a simple and interactive interface for real-time predictions.

## 🚀 Features

* Real-time Loan Approval Prediction
* Interactive Streamlit Web Interface
* Machine Learning-Based Decision Making
* Fast and User-Friendly Application
* Deployment Ready

## 🛠️ Technologies Used

* Python
* NumPy
* Scikit-learn
* Streamlit
* Pickle

## 📊 Input Features

The model uses the following inputs:

* Credit History
* Property Area
* Applicant Income

## ▶️ How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

## 🎯 Prediction Results

The application predicts:

* ✅ Loan Approved
* ❌ Loan Rejected
